LLM-based Business Process Automation
LLM 모델 비교·선정 보고서
1. 비교 목적
본 보고서는 사내 업무 앱 사용설명서 초안 생성 자동화 과업에 적합한 LLM을 선정하기 위해 작성하였다.
비교 대상 모델에는 동일한 업무 과업 프롬프트를 입력하였으며, 각 모델의 결과물을 기준으로 정확성, 형식 준수, 현장 이해도, 실무 활용성, 환각 방지 항목을 평가하였다.
최종 목표는 신규 현장 작업자도 쉽게 이해할 수 있는 MES/WMS PDA 입고관리 앱 사용설명서 초안을 안정적으로 생성할 수 있는 모델을 선정하는 것이다.
---

※ 실제 제출 시 사용 화면에서 정확한 세부 모델명 또는 버전이 다르게 확인되면 수정할 수 있다.

2.1 모델/환경 기록(재현성)
본 비교는 동일한 업무 과업 프롬프트를 각 LLM에 입력한 뒤, 출력 결과를 기준으로 평가하였다.  
재현성을 위해 모델명, 버전, 요금제, 사용 채널, 사용 날짜를 다음과 같이 기록한다.
구분	모델명	버전	요금제	사용 채널	사용 날짜
1	뤼튼 AI	GPT-4	무료	웹	2026-07-05
2	Gemini	Gemini 3.5 Flash	무료	웹	2026-07-05
3	ChatGPT	GPT-5.5 Instant	무료	웹	2026-07-05
주요 설정: 세 모델 모두 웹 무료 환경에서 사용하였으므로 `temperature`, `top_p`, `max_tokens` 등 세부 생성 파라미터는 직접 지정하지 않았다.
무료 버전 사용 시 제한된 점: 무료 버전은 사용 가능 모델, 응답 횟수, 속도, 세부 설정 접근에 제한이 있을 수 있다.
※ 실제 사용 날짜, 요금제, 모델명, 세부 버전이 다를 경우 제출 전 수정한다.
